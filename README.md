# Tecnom Boxes - Frontend Challenge

Aplicación Angular para gestión de turnos de talleres mecánicos.

## Demo

**Frontend**: `http://localhost:4200`  
**API**: `http://localhost:5000/api`

## Ejecutar

```bash
npm install
npm start
```

## Stack Técnico

- **Angular 18** + Standalone Components
- **Angular Material** para UI
- **RxJS** + BehaviorSubjects para manejo de estado
- **Reactive Forms** con validaciones

## Funcionalidades

**Listar turnos** - Tabla con información completa  
**Crear turno** - Formulario multistep con validaciones  
**Validaciones en tiempo real** - Cliente y servidor  
**Manejo de errores** - Feedback visual con snackbars  
**Estados de carga** - UX fluido con spinners  
**Design responsive** - Mobile-first
